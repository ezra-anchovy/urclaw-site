vi. Questions before the call

The first four.

Who owns it, and what happens if you disappear?

You do, and nothing changes. Configs, skills, data, all of it on your hardware or in accounts you control. Nothing lives on our servers. We document the system at install so any competent engineer can pick it up.

Is this an OpenClaw install service?

Sometimes. We also ship on Claude Code, Hermes, and custom agents. We pick what fits your workflow, your compliance posture, and your team.

What's the actual risk of sensitive data leaving our systems?

For high-risk cases we deploy locally with zero cloud calls. Otherwise we document exactly which providers see which data so you can audit it.

What about runaway API bills, or losing my work?

Token caps are set at install. Model spend gets monitored, and broken changes get rolled back. Configs are backed up locally and in your accounts so a Mac Mini wipe doesn't take the system with it.

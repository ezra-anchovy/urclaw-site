# urclaw.com

Public website for UrClaw — a NYC AI operations practice that installs custom AI on client hardware.

Built with vanilla HTML and CSS. Hosted on GitHub Pages, custom domain via CNAME.

## Structure

- `index.html` — homepage (single page)
- `styles.css` — page stylesheet
- `og-image.png` — Open Graph / Twitter card (1200×630)
- `assets/` — M3 mark in bronze and cream-on-dark variants
- `favicon.svg`, `favicon.ico`, `favicon-16.png`, `favicon-32.png`, `apple-touch-icon.png` — branded icons
- `robots.txt`, `sitemap.xml` — crawler basics
- `privacy.html`, `terms.html`, `site.css` — legal pages (legacy styling)

## Deploy

Push to `main`. GitHub Pages auto-deploys to urclaw.com.
